# Traceability Rules

## Zweck
Regeln fuer die Nachverfolgbarkeit von Anforderungen zu Implementierung und Tests.

## Regeln
- Jede FR/NFR verweist auf mind. eine Implementierung in `src/` oder `frontend/`.
- Jede FR/NFR verweist auf mind. einen Test in `tests/`.
- Eintrag in `docs/01_requirements/TRACEABILITY.en.md` ist Pflicht bei Aenderungen.

## Verweise
- `docs/01_requirements/TRACEABILITY.en.md`
- `src/`
- `frontend/`
- `tests/`
