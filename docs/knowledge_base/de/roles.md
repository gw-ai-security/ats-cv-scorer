# Rollen und Verantwortlichkeiten (DE)

## Zweck
Dokumentiere Rollen, die aus dem Repo-Stand ableitbar sind.

## Kerninhalt
- Projektowner: Scope, Ziele, Portfolio-Positionierung.
- Tech Lead: Repo-Struktur, Integration, technische Entscheidungen.
- Software Architect: Architekturentscheidungen und ADRs.
- Requirements Engineer: FR/NFR und Traceability.
- Backend Engineer: Kernmodule (PDF, CV, ATS, JD, Matching).
- Frontend Engineer: Streamlit UI.
- QA Lead/Engineer: Tests, Quality Gates, CI.
- Evaluation Lead: Evaluationsplan und Messungen.
- Learning Engineer: Projekt-Snapshots und Lernartefakte.
